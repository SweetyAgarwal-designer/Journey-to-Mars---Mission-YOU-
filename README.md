# Journey-to-Mars---Mission-YOU-
MISSION: YOU is an immersive, diegetic web experience where users become an astronaut traveling from Earth to Mars. Through scroll, drag, and interactive UI, users navigate launch, deep space, and landing. Guided by an AI companion and cinematic sound, the project blends storytelling, interaction, and emotion into a living mission simulation.

Notes for Best Experience
This is an interactive storytelling experience, not a traditional website.
Please **click on "Soundfield Button" to activate sound and begin the mission.**
Use scroll to progress through the journey from Earth to Mars.
Some sections require interaction (drag, hover, click) — follow on-screen cues. (ex - tapping can trugger ai to explain things)
For the best experience, use desktop/laptop with sound enabled.
Keep volume at a moderate level for cinematic audio immersion.
If interactions seem unresponsive, try refreshing the page.
